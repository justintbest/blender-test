# blender-test

Justin loves Blender. From sculpting organic forms to procedural landscapes, it's become one of his favorite tools for creative exploration. This repository is a collection of scripts and experiments born out of that passion.

## What's here

### `random_ant_landscape.py`
A Blender Python script that generates randomized terrain meshes using fractal noise. Run it once to register a custom **"Landscape"** tab in the 3D viewport N panel, where you can tweak parameters like height, noise scale, octaves, and mesh size — then hit Generate to create a new landscape.

## Requirements
- Blender 4.2+

## Usage
1. Open Blender and go to the **Scripting** workspace
2. Open `random_ant_landscape.py`
3. Run the script (**Alt+P**)
4. Switch to the **3D Viewport**, press **N**, and open the **Landscape** tab
